# Whop Automation

## How It Works
Whop payment data is automatically fetched and updated via GitHub Actions.

## Data
- File: `whop-payments.json`
- Contains full payment records with membership, product, user, and billing details
- Auto-updated via GitHub Actions workflow

## Fields Tracked
- Payment ID, status, amounts (subtotal, final, refunded)
- User email and username
- Product and membership IDs
- Card details (brand, last 4)
- Billing address
- Affiliate attribution
- Acquisition data
