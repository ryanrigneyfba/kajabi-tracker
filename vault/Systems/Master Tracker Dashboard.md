# Master Tracker Dashboard

## Overview
Single-page React dashboard hosted on GitHub Pages that aggregates all revenue data.

## URL
[Stay Viral Master Tracker](https://ryanrigneyfba.github.io/kajabi-tracker/)

## Tech Stack
- React 18 (CDN, no build step)
- Tailwind CSS (CDN)
- Babel standalone (in-browser JSX)
- Single `index.html` file

## Features
- Combined revenue across Kajabi, TikTok Agency, and Whop
- Info Profit (Kajabi + Whop)
- Agency Profit (TikTok, with 15% manager deduction)
- Combined Profit view
- Dark theme with cyan accent
- Mobile responsive

## Data Sources
Dashboard fetches directly from the repo's JSON files:
- `transactions.json` — Kajabi sales
- `agency-data.json` — TikTok agency payouts
- `whop-payments.json` — Whop payments
